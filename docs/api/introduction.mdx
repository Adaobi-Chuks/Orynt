## üìù Making Requests

All requests must include:

- `Content-Type: application/json`
- `X-Network` header for your chosen network
- `X-PAYMENT` header if the request requires payment (HTTP 402 response)

**Example Request: Generate an image using DALL¬∑E 3**

```bash
curl -X POST "https://api.oryntai.xyz/api/v1/ai/images" \
  -H "Content-Type: application/json" \
  -H "X-Network: solana" \
  -H "X-PAYMENT: <your_payment_signature>" \
  -d '{
    "model": "dall-e-3",
    "prompt": "A futuristic city skyline at sunset",
    "n": 1,
    "width": 1024,
    "height": 1024
  }'
```
