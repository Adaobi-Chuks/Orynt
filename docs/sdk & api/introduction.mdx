## üì¶ SDK

The **Orynt SDK** is model-agnostic, offering simple, unified access to AI capabilities including chat, embeddings, image generation, text-to-speech, and speech-to-text.

**SDK Details:**

- **Name:** `orynt/ai-x402`
- **Installation:**

```
npm install orynt-sdk
```

- **Features:**
  - Unified AI Endpoints for multiple AI tasks
  - Model-Agnostic: Works with OpenAI, Gradient, and more
  - Pay-per-Request: Handles USDC micropayments via x402
  - Wallet-based Authorization: No API keys required

## üåê REST API

The REST API mirrors all SDK functionality. Ideal for projects outside Node.js environments.

Users can send direct API requests via the base URL:

```
https://api.oryntai.xyz/api/v1
```

**Example Request**

```
    curl -X POST https://api.oryntai.xyz/api/v1/ai/images \
      -H "Content-Type: application/json" \
      -H "X-Network: solana" \
      -d '{
        "model": "dall-e-3",
        "prompt": "A futuristic city skyline at sunset",
        "n": 1,
        "width": 1024,
        "height": 1024
      }'
```
