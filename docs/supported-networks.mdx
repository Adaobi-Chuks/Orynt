# üåê Supported Networks & Tokens

## üß© Example Integration

**SDK Example:**
```javascript
import { Orynt } from '@orynt/ai-x402';

const ai = new Orynt({
  network: 'solana-devnet' // default if not provided
});

await ai.chat({
  model: 'gpt-4.1',
  messages: [{ role: 'user', content: 'Hello' }]
});
```

**REST Example:**
```bash
curl -X POST "https://api.oryntai.xyz/api/v1/ai/images" \
-H "Content-Type: application/json" \
-H "X-Network: solana" \
-H "X-PAYMENT: <your_payment_signature>" \
-d '{
  "model": "dall-e-3",
  "prompt": "A futuristic city skyline at sunset",
  "n": 1,
  "width": 1024,
  "height": 1024
}'
```
